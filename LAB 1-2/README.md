# Task 1: Three-Line Welcome Message
## Overview

This program prints a three-line welcome message using separate cout statements. Each line is displayed on a new line using endl.

## Objective

To practice using multiple cout statements and using endl to move to the next line.

## Output

Welcome to the C++ lab!<br>We will learn basic output today.<br>Let's start with simple messages.

## Learning Outcomes

Use multiple cout statements correctly

Understand how endl creates a new line

Practice structured console output

# Task 4: Displaying Quotes, File Paths, and New Lines
## Overview

This program demonstrates how to correctly display:

Quotation marks inside a string

A Windows file path containing backslashes

Text on multiple new lines using escape sequences

## Objective

To help beginners understand the use of escape characters in C++ such as:

\" for quotes

\\ for backslashes

\n for new lines

## Output

She said, "Let's Learn C++!"<br>To start, save your file in C:\C++_Projects\Lab 1<br>Good Luck!

## Learning Outcomes

Use escape sequences in C++ correctly

Print quotes and file paths without errors

Format console output with proper line breaks
